Sublime Taskwarrior

A plugin for ST2 to view, modify and add tasks with Taskwarrior.

Commands

super+alt+t - View tasks

super+shift+a - Add task

Dependencies

taskw

Taskw is a python API for Taskwarrior.

If you don't have pip-2.6: sudo easy_install-2.6 pip

Then: sudo pip-2.6 install taskw
